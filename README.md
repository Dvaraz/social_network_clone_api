**Основной функционал**
+ Регистрация с потвержжением по e-mail
+ Авторизация по jwt token
+ Редактирование профиля
+ Создание постов
+ Просмотр постов
+ Лайки для постов
+ Уведомление для лайков
  + Остольные уведомления в процессе
+ Добавление друзей
+ Чат
+ Поиск пользоваетлей
+ Поиск постов по словам, по автору поста
+ Тренды
